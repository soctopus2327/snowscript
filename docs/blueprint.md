# **App Name**: SnowScript: Winter of Code

## Core Features:

- Homepage Hero Section: Display a large title, GDG logo, and a 3D snowflake iframe.
- Registration Button: A prominent 'Register Now' button on the homepage.
- About GDG Section: A card with text, an image, and buttons linking to WhatsApp, Discord, and Instagram groups.
- Contribution Guide: A 'How to Contribute' section outlining the contribution process.
- Event Timeline: A visual timeline displaying important dates and milestones.
- FAQ Section: An expandable FAQ section answering common questions.
- Projects Page: Display cards for 8 project domains (AR/VR, AI/ML, AppDev, WebDev, DSA, CP, Technical Writing, UI/UX).
- Project Domain Exploration: On hovering on a project card display, provide an interface with a button to display sub-project cards
- Prizes Page: A page showcasing prizes in different categories.
- Snow Effect: Add a falling snow effect to enhance the winter theme.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), evoking a sense of winter and coding.
- Background color: Light, desaturated blue (#D3DCE6), providing a cool and calming backdrop.
- Accent color: Soft yellow (#FDE68A), used sparingly for highlighting key elements such as CTAs.
- Body text: 'Inter', a neutral, grotesque-style sans-serif font that works well for both headlines and body text.
- Note: currently only Google Fonts are supported.
- Use clean, simple icons representing each project domain and FAQ category.
- Employ a card-based layout for project domains, projects, and FAQs to maintain organization and visual appeal.
- Implement subtle hover animations for project cards and interactive elements to provide feedback.
- Use the falling snow animation subtly to avoid overwhelming the user experience.
